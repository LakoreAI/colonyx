# Scripts

Release helpers for local validation and publishing.

## Available scripts

- `scripts/check-release.sh` — run local build and packaging checks.
- `scripts/build-release.sh` — build the Rust extension and Python wheel artifacts.
- `scripts/publish-pypi.sh` — upload wheel/sdist artifacts to PyPI.
- `scripts/publish-crates.sh` — publish the Rust crate to crates.io.

All scripts are intended to be run from the repository root.
